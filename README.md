# PatternGen

PatternGen is a desktop app for broadcast and live production that generates PNG images in two modes: **Slates** and **Test Patterns**. Slates are labeled color cards used to identify cameras or inputs, with support for single or batch generation. Test Patterns include SMPTE color bars, alignment grids, boundary lines, and more — all composable and exportable at resolutions up to 16K.

## Installation

**Requirements:** [Node.js](https://nodejs.org) (includes npm)

```bash
git clone https://github.com/your-username/slategen.git
cd slategen
npm install
npm start
```
